Bootstrap Button Generator
The simplest way to generate new button styles for Twitter's Bootstrap

How to use BBG

It's easy. Really. Just enter the css class you want to assign to your new button style and play with the different button colors to update the live previews in the left.

When you are ready, copy the code below and add it to your page below bootstrap.min.css to use the button.
